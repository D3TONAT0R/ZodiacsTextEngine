The ZodiacsTextEngine is a simple library for creating text adventures, originally created for the game Sailing Zodiacs. Content for the game is written using a custom text format, intended to be easy to write, needing only minimal amounts of special characters. The engine can be integrated into any .NET application that can display text and handle user input, ranging from simple console applications to game engines like Unity.




Features

  • Simple room file system.
  • Multi-color text output.
  • Includes a pre-built console application that can load custom content out of the box.
  • Support for variables and conditions for branching paths.
  • Ability to write custom functions to extend the engine's functionality.
