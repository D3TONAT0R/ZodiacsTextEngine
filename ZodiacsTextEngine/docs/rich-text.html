Rich Text

Rich Text is a text format used by the +TEXT effect that uses tags to apply different effects to parts of a text, such as styling, displaying a variable, or showing the output of a custom function.

Example

The <color=yellow>quick brown fox<clear> jumps <background=blue><color=red>over<clear> <var=dogs> lazy dogs. The quick brown fox jumps over 3 lazy dogs.

Available Tags

Tag Effect
<color=COLOR> Changes the text color. Available colors are: red, green, blue, yellow, magenta, cyan, and white.
<background=COLOR> Changes the background color. Available colors are the same as for text color.
<clear> Resets all styles to default.
<var=VARIABLE_NAME> Displays the value of a variable.
<svar=VARIABLE_NAME> Displays the value of a string variable.
<func=FUNCTION_NAME> Calls a custom function and displays its output. Parameters can be passed by appending them in parentheses directly after the function name, separated by commas.
e.g. <func=FUNCTION_NAME(1,2,3)>
