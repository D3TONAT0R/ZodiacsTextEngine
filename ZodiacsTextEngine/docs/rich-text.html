Rich Text

Rich Text is a text format used by the +TEXT effect that uses tags to apply different effects to parts of a text, such as styling, displaying a variable, or showing the output of a custom function.

Example

The <color=yellow>quick brown fox<clear> jumps <background=cyan><color=red>over<clear> <var=dogs> lazy dogs. The quick brown fox jumps over 3 lazy dogs.
The text inside a tag (between '<' and '>') must not contain any spaces, otherwise it will not be parsed.

Available Tags

Tag Effect
<color=COLOR> Changes the text color. See below for a list of available colors.
<c=COLOR> Shorthand for <color=COLOR>.
<background=COLOR> Changes the background color. See below for a list of available colors.
<b=COLOR> Shorthand for <background=COLOR>.
<h> Applies highlighting to the text as defined in the engine settings. Both the text and the background color can be changed.
<reset> Resets all styles to default.
<r> Shorthand for <reset>.
<var=VARIABLE_NAME> Displays the value of a variable. The value can be optionally be formatted in the following ways:
Appending *x will multiply the output value by x.
Appending @x will format the output to x decimal places.
Appending :x will format the output using .NET numeric formats , where x is the format specifier.
(@x and :x are mutually exclusive)

Example: <var=score*12.345@2> will display the value of the variable score multiplied by 12.345 and formatted to 2 decimal places.
<svar=VARIABLE_NAME> Displays the value of a string variable. The case of the output can be changed by:
Appending @U will convert the output to uppercase.
Appending @L will convert the output to lowercase.
<func=FUNCTION_NAME> Calls a custom function and displays its output. Parameters can be passed by appending them in parentheses directly after the function name, separated by commas.
e.g. <func=FUNCTION_NAME(1,2,3)> will invoke the function with three parameters: 1, 2, and 3.

Colors

The following colors are available for use in rich text and in color effects, matching .NET's ConsoleColor Enum:

  • black
  • blue
  • cyan
  • dark_blue
  • dark_cyan
  • dark_gray
  • dark_green
  • dark_magenta
  • dark_red
  • dark_yellow
  • gray
  • green
  • magenta
  • red
  • white
  • yellow
